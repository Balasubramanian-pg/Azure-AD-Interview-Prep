# Role-Based Access Control (RBAC)

## Azure AD Roles
- **Global Administrator**: Full access to all features
- **User Administrator**: Manage users/groups
- **Application Administrator**: Manage app registrations
- **Security Administrator**: Manage security policies
- **Conditional Access Administrator**: Manage CA policies

## Azure Resource Roles
- **Owner**: Full access + manage roles
- **Contributor**: Full access (no role management)
- **Reader**: Read-only
- **User Access Administrator**: Manage user access

See azure-cli-commands.sh for RBAC command examples.
