# Azure Active Directory

## Key Differences: AD vs Azure AD
- Protocol: LDAP/Kerberos vs OAuth 2.0/OIDC/SAML
- Scope: On-premises vs Cloud-based
- Structure: OUs/GPOs vs Flat structure
- Authentication: NTLM/Kerberos vs Modern auth (MFA, CA)

## Azure AD Editions
- **Free**: Basic user/group management, SSO, MFA
- **P1**: Conditional Access, Hybrid Identity, Dynamic Groups
- **P2**: Identity Protection, PIM, Risk-based policies

See commands.ps1 for Azure AD PowerShell examples.
